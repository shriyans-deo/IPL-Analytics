# IPL-Analytics
Built an interactive Excel dashboard using Pivot Tables, Pivot Charts, and Slicers to analyze and visualize data from multiple datasets efficiently.
Developed a dynamic and interactive Excel dashboard using Pivot Tables, Pivot Charts, and Slicers to analyze and visualize large datasets efficiently. Integrated multiple data sources through the Data Model to enable synchronized filtering and comparative insights across charts. The project focused on data cleaning, relationship management, and creating user-friendly visual analytics for better decision-making and trend analysis.
